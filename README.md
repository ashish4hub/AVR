# AVR
THIS REPOSITORY CONTAINS **bare metal AVR programming** using **Avr-GCC** and **direct register level access**.
---
## Toolchain
- Language: **C**
- Compiler: **Avr-GCC**
- Programmer:**USBasp**
- MCU: **ATmega328P**
---
## Examples
- Blinking led using **ATmega328P** and **register level programming (bare metal)** with a delay of 1 second.
---
## UART (USART)
- Interrupt driven Rx using ring buffer
- Interrupt driven Tx using ring buffer
- Non blocking serial communication
- Hardware timer based
